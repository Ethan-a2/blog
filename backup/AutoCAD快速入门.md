# 最佳实践
- 使用其它人画好的图块，跟sketch up组件的概念类似。会节省大量的时间。
- 勤用空格：按一次结束上一命令，再按一次重复上一次命令。方便连续画线。
- 多个线段选中后输入"join"会合成一个整体。
- 一个整体选"爆炸"可以切分成多个线段


# 遇到问题及解决
- 无法剪切，提示：不与剪切边相交。肉眼看着是相交了，但是AutoCAD非说不相交。那就把线段移一下，强制明显的相交，这样就能剪切了。
